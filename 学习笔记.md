# 创建项目
安装 npm install -g create-react-app 
检查是否安装成功 create-react-app -V
创建react项目 npx create-react-app <my-project>
cd my-project
npm start

node 版本16.16.0

注意：
1.组件首字母必须大写
# 父子通信 受控组件 
# 类组件 rcc
# hooks rfc
useState 保存组件状态
useEffect 没学明白
useCallback 记忆函数
useMemo 经过函数计算一个值，（相当于 vue 计算属性）
useRef 保存引用值
useReducer useContext (减少组件层级)
